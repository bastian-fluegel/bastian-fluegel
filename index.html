Hinweis zum US-Hosting

Warum wird diese Seite in den USA gehostet?
Diese temporäre "Under Construction"-Seite wird über GitHub Pages bereitgestellt – ein kostenloser und zuverlässiger Dienst von GitHub Inc. mit Sitz in den USA.

Transparenz ist mir wichtig: Als Security-Spezialist weiß ich, dass US-Hosting datenschutzrechtliche Fragen aufwirft. Deshalb hier die Fakten:

Was bedeutet das für Ihre Daten?
Beim Aufruf dieser Seite werden technische Daten (IP-Adresse, Browsertyp, Zeitstempel) von GitHub-Servern in den USA verarbeitet. Dies ist technisch unvermeidbar und erfolgt auf Basis meines berechtigten Interesses (Art. 6 Abs. 1 lit. f DSGVO).

Wie ist das DSGVO-konform?
GitHub ist unter dem EU-US Data Privacy Framework zertifiziert. Dieses Framework wurde von der EU-Kommission als angemessenes Schutzniveau anerkannt und ermöglicht rechtmäßige Datenübertragungen in die USA.

Meine finale Website
Die vollständige bastian-fluegel.de wird auf deutschen Servern (Hetzner, Gunzenhausen) gehostet – ohne US-Datenübertragung, mit A+ Security-Rating und strikter DSGVO-Compliance.
